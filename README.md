# Python Environment Manager

A Python tool designed to automate virtual environment setup and dependency management. This project aims to simplify and stabilize Python development workflows by ensuring consistent environments across machines.

## Features

- Automated creation and activation of virtual environments  
- Dependency conflict detection and resolution  
- Integration with pip and Poetry for package management  
- Environment consistency checks and updates

## Tech Stack

- Python 3.x  
- Poetry  
- pip  
- GitHub Actions (for CI/CD)
